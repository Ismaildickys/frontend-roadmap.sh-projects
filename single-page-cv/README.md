# HTML CV Project

A simple and responsive curriculum vitae (CV) webpage built using HTML.
Easily customizable and accessible in any web browser.

## Getting Started

### Prerequisites
- A web browser (Chrome, Firefox, Safari, etc.)
- A text editor (VS Code, Sublime Text, Atom, etc.)

### Running the CV
1. **Clone the repository** (if applicable):
   ```bash
   git clone https://github.com/ismaildickys/projects-roadmap.git
   ```

2. **Open the HTML file**:
   - Navigate to the project directory
   - Double-click `index.html` to open it in your default browser

   **OR**

   - Right-click `index.html` → "Open With" → Choose your preferred browser

3. **View the CV**:
   - The CV will render in your browser
   - Use browser zoom if needed for better readability

### Editing the CV
1. Open `index.html` in your text editor
2. Modify the content within the HTML tags
3. Save changes and refresh your browser to see updates

## Customization
- **Content**: Edit text directly in the HTML file
- **Styling**: Add a `<style>` section or link a CSS file for custom styling
- **Sections**: Modify or add new sections by editing the HTML structure
